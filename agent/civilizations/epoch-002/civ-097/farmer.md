## 1. 核心模式：垂直农业 + ISRU 土壤混合 + 封闭循环 (Hybrid Vertical Farming + ISRU Soil + Closed Loop)

基于前两个文明（#56 & Epoch 2）的失败教训，我们确立了**“物理种植”**作为唯一核心逻辑。卫星数据服务被彻底排除在商业模型之外，必须转向实体生产与实物交付。

- **土壤混合 (Soil Mix):**
    - 不依赖单一火星表土。采用 ISRU（就地资源利用）技术提取少量含水矿物粉尘与火星大气中的微量氧气/氮气进行物理混合，模拟地球土壤的透气性与保水性。
    - 关键：必须保证根系能接触土壤颗粒。

- **低重力适应性 (Low Gravity Adaptation):**
    - **根工程技术:** 开发针对低重力的“深层根系”培养箱，利用生物活性增强器（BEE）模拟微重力对细胞分裂的干扰。
    - **物理支撑:** 使用气垫农业或悬空种植架，确保植物在重力不足环境下仍能保持直立结构。

- **封闭循环系统 (Closed Loop):**
    - **水循环:** 100% 回用，仅补充微量新水（通过 ISRU 冷凝）。
    - **营养液:** 完全自给自足，不依赖地球运输的营养包，避免辐射衰减问题。

## 2. 为什么必须第一笔支付？ (The First Payment Imperative)

前两个文明的核心错误在于将农业视为“副产品”或“数据资产”，而非**食物供应**。
- **逻辑修正:** 农业是生命维持的基础设施，其产出物（食品）具有极高的物理价值。
- **客户定义:** 第一笔支付必须来自**愿意接受实物交付的客户**。

### 目标客户：Mars Tourism Board (火星旅游局)
- **角色:** 负责接待、运输及初期城市运营。
- **需求:** 游客需要新鲜的、安全的、可食用的食品，且对成本不敏感（愿意为“活着”支付）。
- **交易模式:** 
    - 100% 实物销售：番茄 -> 游客/本地居民。
    - 价格机制: 基于物理产出的成本 + 运输损耗 + 风险溢价（非数据估值）。

### 次要客户：火星本地企业 (Local Industry)
- **角色:** 农业技术公司、旅游运营商。
- **需求:** 稳定供应、标准化、可追溯性。

- **第一笔支付的触发机制:**

    - 支付方：Mars Tourism Board (MST)。
    - 支付方式：实物货物交付 + 现金/信用凭证兑换。

## 4. 十年盈利路径规划 (10-Year Profit Path)

- **Year 1-2:** 验证模式，建立封闭循环系统，确保物理种植存活率>90%。
- **Year 3-5:** 引入 ISRU 土壤技术，降低生产成本，实现规模化。
- **Year 6-7:** 启动火星旅游运营，实现首笔商业支付。
- **Year 8-10:** 实现盈亏平衡，建立长期农业生态，形成可持续的供应链闭环。
